# Tenant-Rent-Record-System-C++
A simple console-based Tenant Rent Management System developed using C++. This project allows users to manage tenant records including storing, searching, updating, and deleting rent information.

## Features
- Add new tenant records  
- Display all tenant records  
- Search tenant by Tenant ID  
- Update tenant information  
- Delete all tenant records  
- Menu-driven console interface  

## Technologies Used
- C++  
- Object-Oriented Programming (OOP) concepts  
- Standard Template Library (STL) (`iostream`, `string`)
